\btext{.50}{.30}{.40}{\large Main:\\
In the main, there will be only communication part.\\
Other works will be handled in separte functions.\\~\\
First, the program will ask for the dictionary file\\
name and then store the data into linked list\\
(loadDictionary fun.)\\
After that, it will ask for which words that users\\
want to search.\\~\\
If the word doesn't exist in the data,\\
it will ask for creating a new word.\\
If users create the word,\\
then it will be stored into the file.\\
It repeats the operations in a while loop until\\
users hit ctrl+D.}
